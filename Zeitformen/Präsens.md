## Regelmäßige Verben

|           | Präsens Bildung |
| --------- | --------------- |
| ich       | geh*e*          |
| du        | geh*st*         |
| er/sie/es | geh*t*          |
| wir       | geh*en*         |
| ihr       | geh*t*          |
| sie/Sie   | geh*en*         |

## Unregelmäßige Verben
|           | Präsens Bildung |
| --------- | --------------- |
| ich       | bin             |
| du        | bi*st*          |
| er/sie/es | *ist*           |
| wir       | *sind*          |
| ihr       | *seid*          |
| sie/Sie   | *sind*          |
