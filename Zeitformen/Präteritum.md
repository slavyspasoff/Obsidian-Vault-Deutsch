## Bildung
### regelmäßige Verben
|           | Präteritum Bildung |
| --------- | ------------------ |
| ich       | lieb*te*           |
| du        | lieb*test*         |
| er/sie/es | lieb*te*           |
| wir       | lieb*ten*          |
| ihr       | lieb*tet*          |
| sie/Sie   | lieb*ten*          |
### unregelmäßige Verben
|           | Präteritum Bildung |
| --------- | ------------------ |
| ich       | *lief*             |
| du        | lief*st*           |
| er/sie/es | *lief*             |
| wir       | lief*en*           |
| ihr       | lief*t*            |
| sie/Sie   | lief*en*           |

### Mischverben

|           | Präteritum Bildung |
| --------- | ------------------ |
| ich       | w*u*ss*te*         |
| du        | w*u*ss*test*       |
| er/sie/es | w*u*ss*te*         |
| wir       | w*u*ss*ten*        |
| ihr       | w*u*ss*tet*        |
| sie/Sie   | w*u*ss*ten*        |
## Beispiele
- Damals *fuhr* ich mit dem Rad zur Schule.
- Könige Viktoria *regierte* mehr als 60 Jahre über Großbritannien.
